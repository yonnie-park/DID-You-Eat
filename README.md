# DID You Eat?
![image](https://user-images.githubusercontent.com/77462765/217274743-d2188172-ed22-4312-a140-14f2f1898235.png)

DID YOU EAT allows customers to collect unique, one-of-a-kind soul bound tokens from the restaurants they visit. These SBTs serve as a digital keepsake, capturing the memories and experiences of each dining adventure.
